# OIBSIP Task 3

## Project Name
Car Price Prediction with Machine Learning

## Objective
To build a Machine Learning model that predicts the selling price of a car based on features such as present price, kilometers driven, fuel type, selling type, transmission, and ownership.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
Car Price Prediction Dataset

## Machine Learning Models Used
- Linear Regression
- Random Forest Regressor

## Results
- Linear Regression R² Score: 0.8467
- Random Forest R² Score: 0.9656
- Predicted Car Price (Sample): 3.84 Lakhs

## Features Used
- Year
- Present Price
- Driven Kms
- Fuel Type
- Selling Type
- Transmission
- Owner
