# OIBSIP Task 5

## Project Name
Sales Prediction Using Python

## Objective
To predict product sales based on advertising expenditures on TV, Radio, and Newspaper using Machine Learning techniques.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
Advertising Dataset

## Machine Learning Models Used
- Linear Regression
- Random Forest Regressor

## Features Used
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

## Results
- Linear Regression R² Score: 0.8994
- Random Forest R² Score: 0.9813
- Mean Absolute Error (MAE): 0.6201
- Predicted Sales (Sample): 21.92
