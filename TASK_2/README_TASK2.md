# OIBSIP Task 2

## Project Name
Unemployment Analysis with Python

## Objective
To analyze unemployment trends across different regions of India and study the impact of COVID-19 on unemployment rates using data analysis and visualization techniques.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
- Unemployment in India Dataset

## Analysis Performed
- Data Cleaning and Preprocessing
- Missing Value Handling
- Statistical Summary
- Unemployment Rate Distribution Analysis
- Region-wise Unemployment Analysis
- Monthly Trend Analysis
- Correlation Heatmap
- COVID-19 Impact Analysis

## Key Insights
- Tripura recorded the highest average unemployment rate.
- A significant rise in unemployment was observed during the COVID-19 period.
- The overall average unemployment rate was approximately 11.79%.
